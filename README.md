# AHU-SI-Repository

安徽大学互联网学院资源仓库

### 为什么要创建这个仓库？

受相关项目启发，为互联网学院的同学们汇总资料、提供一个学习交流的平台。希望本仓库不仅有课程相关的资料，更能有一些类似各高校**飞跃手册**、**生活指南**的参考文档。

### 怎么使用这个仓库？

确保网络畅通。并确保你安装了[Git](https://git-scm.com/)

使用git克隆最新版本
```
git clone --depth 1 https://github.com/hermit403/AHU-SI-Repository
```

如果你还是不会使用，可以尝试**复制你终端中的报错信息，并粘贴给deepseek/kimi/qwen等任一AI聊天工具**，并询问“为什么会报错，如何解决？”

### 为什么现在仓库很空？

事实上，已经有一些同类项目收录了较为全面的资料了。很多试卷完全可以在这些repo获取。因此本项目还在架构阶段：**repo的重心将会放在指南类文档**上，并力求向未入学及刚入学的同学们提供一个客观的介绍。如果你有任何对于学院的疑问，欢迎联系我。

tips: 资料收集可以由AI驱动，但遵循**Rules**所述，要有人类参与审计

### The future of this Repo

- 提供更为便捷的阅读、使用方式
- 持续丰富本repo，为同学们提供帮助
